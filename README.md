# halo-cli

## 安装

```bash
# 使用npm安装CLI
$ npm install https://github.com/infinitas-plus/halo-cli.git -g

# 使用yarn安装CLI
$ yarn global add https://github.com/infinitas-plus/halo-cli.git
```
